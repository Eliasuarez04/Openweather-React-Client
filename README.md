A fast and responsive weather application that fetches and displays real-time meteorological data for any city worldwide, demonstrating asynchronous JavaScript and RESTful API integration.

🔗 **[Live Demo: Try the App Here](https://weather-app-inky-six-45.vercel.app/)**

## ✨ Key Features
* **Real-Time Data:** Integrates with the OpenWeather API to fetch live weather conditions (temperature, humidity, wind speed, etc.).
* **Dynamic Search:** Users can search for any city globally with error handling for invalid queries.
* **Optimized Build:** Bootstrapped with Vite for instant loading and development speed.
* **Responsive UI:** Clean interface that adapts perfectly to both mobile devices and desktop screens.

## 🛠️ Tech Stack
* **Frontend:** React.js, JavaScript (ES6+), HTML5, CSS
* **Build Tool:** Vite
* **External Services:** OpenWeather REST API

## 🚀 Installation & Local Setup

To run this project locally, you will need a free API key from OpenWeather.

1. Clone the repository:
   ```bash
   git clone [https://github.com/Eliasuarez04/Openweather-React-Client](https://github.com/Eliasuarez04/Openweather-React-Client)

2. Navigate to the directory:
   cd weather-app

3. Install the dependencies:
   npm install

4. Environment Variables: Create a .env file in the root directory and add your OpenWeather API key:
   VITE_WEATHER_API_KEY=your_api_key_here

5. Start the development server:
   npm run dev
